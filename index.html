About Leo


Born and raised in the Tri-State area, I moved to Portland where I not only fell in love with the city, but also found a passion for coding and web design.

As a recent graduate of The Tech Academy, I am now a trained Junior Developer and am experienced in HTML, CSS, JavaScript and C#. Some of my recent projects include building a basic Car Insurance MVC webapp, created using ASP.net and Entity Framework, a pizza ordering web page, created using JavaScript, and contributed to an MVC Web App for a local theater group using Entity Framework Code First.
