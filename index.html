About Leo


Born and raised in the Tri-State area, Leo moved to Portland where he not only fell in love with the city, but also found a passion for coding and web design.

A recent graduate of The Tech Academy, Leo is now a trained Junior Developer and is proficient in HTML, CSS, Javascript and C#. Some of his recent projects include building a basic Car Insurance MVC webapp, created using ASP.net and Entity Framework, a pizza ordering app, created using Javascript, and contributed to a website for a local theater group created with Entity Framework Code First.
